Quiz4TV

Un completo juego de preguntas tipo Trivial para Chromecast, desarrollado con el SDK de Google Cast que comunica la aplicación Android con el servidor del juego construido en AngularJS que se nutre de preguntas mediante una API REST de un backend desarrollado con Spring Framework y MySQL.

fjalonso.me

Este sitio web está desarrollado usando Jekyll como gestor de contenidos estático, como tal no depende de ninguna base de datos, utiliza ficheros de texto para almacenar los contenidos, datos y configuración. Para construir la interfaz de usuario se ha utilizado básicamente el framework Foundation con Sass y jQuery.
